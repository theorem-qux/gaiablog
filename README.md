# 尾須浦のブログ — 開発メモ

ARG用静的サイト。GitHub Pagesでホスティング予定。

## デプロイ前にやること

1. `sitemap.xml` と `robots.txt` 内の `[YOUR-USERNAME]` と `[REPO-NAME]` を実際のGitHubユーザー名・リポジトリ名に置換する。
2. `index.html` の OGP `og:url` タグにも同様に本番URLを設定する（任意）。

```
blog/

```

## 記事ファイルの作り方

`articles/article-template.html` をコピーして `no-XXX.html` にリネームし、
ファイル末尾の CHEATSHEET を参考に内容を書き換える。

### カテゴリB（呪われた記事）の注意点

`.article-body` を `.cursed-body` クラスのdivに差し替え、
中身を `0.362412015384720` の繰り返しで埋める。
